DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/
  PyGame:     http://www.pygame.org/

  or dubble click "run_game.exe"



RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.exe" file and double-click it.

Otherwise open a terminal / console and "cd" to the game directory and run:
  python run_game.py



HOW TO PLAY THE GAME:

Control the flow of water by arrow keys
Make sure to use them wisely because the trash must be collected
in the bucket and not stuck in between stones
